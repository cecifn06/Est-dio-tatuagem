Landing page responsiva para um estúdio de tatuagem fictício, criada como exercício prático de desenvolvimento front-end. O objetivo do projeto foi reforçar conhecimentos em HTML5, CSS3, responsividade, animações e estruturação de interfaces modernas.

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- AOS (Animate On Scroll)
- Font Awesome
- Google Maps Embed
